Random Hough Forest
===================
Object recognition and learning library

# Requirements
Following programs are requrd to build:

 - gcc >= 4.6.3 with C++ support
 - CMake >= 2.8
 - OpenCV >= 2.4.6
 - Boost >= 1.5

# Recommend
 - OpenMP

# Installation

  $ cmake .
  $ make
  $ sudo make install
